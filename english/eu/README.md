# Civic Forensics — EU English Edition

This folder contains English-language Civic Forensics prompts adapted for use in EU Member States and for matters involving EU institutions, EU law, EU funding and multi-level governance.

Author: **Velimir Samara**  
Initiative: **Civic Forensics / Građanska forenzika**  
Licence: **Creative Commons Attribution 4.0 International (CC BY 4.0)**

## Files

### Quick citizen analysis

- `quick-citizen-analysis-prompt.txt` — shorter prompt for citizens analysing a news report, public statement or official announcement.
- `quick-citizen-analysis-short-user-guide.txt` — concise practical instructions for EU citizens.

### Full public statement analysis protocol

- `public-statement-analysis-user-guide.txt` — detailed user guide for the full protocol.
- `public-statement-analysis-protocol/part-01.txt`
- `public-statement-analysis-protocol/part-02.txt`
- `public-statement-analysis-protocol/part-03.txt`
- `public-statement-analysis-protocol/part-04.txt`
- `public-statement-analysis-protocol/part-05.txt`

The five protocol parts form one continuous prompt and should be copied in numerical order into the AI tool.
