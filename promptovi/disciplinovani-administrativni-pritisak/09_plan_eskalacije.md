# 09 — Plan eskalacije po stepenima

## Kada koristiti

Koristite ovaj prompt kada imate više dopisa, odgovora ili rokova, ali problem nije riješen. Cilj je napraviti uredan plan: šta je sljedeći zakonit korak, kome se šalje i koji dokaz se koristi.

Eskalacija ne treba biti emotivna. Treba biti dokumentovana.

## Prompt

```text
Na osnovu dosadašnje prepiske napravi plan eskalacije po stepenima.

Materijal:
[OVDJE ZALIJEPI KRATKU HRONOLOGIJU, ODGOVORE ORGANA ILI SAŽETAK]

Zadatak:
1. izdvoji dosadašnje ključne korake;
2. utvrdi šta je organ odgovorio, a šta nije;
3. predloži sljedeće korake po redu, od blažeg ka jačem;
4. za svaki korak napiši kome se šalje, cilj, dokaz koji se koristi i šta se traži;
5. uključi mogućnosti: dopuna prijave, FOI zahtjev, urgencija, pritužba rukovodiocu, prijava upravnoj inspekciji, predstavka ombudsmanu, javna analiza ili pravni lijek ako postoji osnov;
6. ne predlaži uvredljiv, prijeteći ili politički ton;
7. označi koje korake treba provjeriti sa pravnikom ako imaju pravne posljedice.

Rezultat organizuj u tabeli:
- redni broj;
- sljedeći korak;
- kome se šalje;
- cilj;
- dokaz koji se koristi;
- očekivani rezultat;
- rizik ili napomena.
```

## Kako koristiti rezultat

Plan eskalacije koristite kao mapu, ne kao automatsku naredbu. Neke korake možda treba preskočiti ili prilagoditi zavisno od predmeta.

## Pravilo

Svaki novi korak treba koristiti prethodni dokument kao dokazni trag.
